# korg-tr-ableton-live
Korg TR keyboard configuration for the DAW Ableton Live.

This configuration bind the **Korg TR** knobs to the instrument rack macros on **Ableton Live** for realtime control.

## **Knob midi map**

* CUTOFF ---------> **CC74**
* RESONANCE ----> **CC71**
* EG-INTENSITY --> **CC79**
* ES-RELEASE -----> **CC72**
* ASSIGNABLE 1 --> **CC73**
* ASSIGNABLE 2 --> **CC19**
* ASSIGNABLE 3 --> **CC20**
* ASSIGNABLE 4 --> **CC21**
